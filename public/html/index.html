¿Qué es Floppy?

Floppy es un sistema desarrollado para la agilización de apertura y cierre de portones de manera automática, delimitado a fraccionamientos con baja afluencia y para la región única del Estado de México, México. Es exclusivo para fraccionamientos con acceso único a automóviles. Así mismo cuenta con una página web donde el habitante podrá hacer consulta sobre el estado de su automóvil, refriéndonos a si se encuentra dentro o fuera del estacionamiento.
